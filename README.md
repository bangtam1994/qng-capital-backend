## Description

QNG Capital backend in nest, using typeorm and postgres db.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn dev

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
